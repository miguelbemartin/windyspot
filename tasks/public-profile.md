# Task: Create a public profile page

## Goal
The goal is that users can have their own public profile, so they can share with others.

## Requirements
- page should not be protected by login, so it should be accessible without account
- page should use a slug so it's SEO friendly
- user can have a default slug, and they can update it later on
- slug should be unique
- slug should be validated to it does not contain characters that are not appropiate for URLs
